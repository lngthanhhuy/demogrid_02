# Art

Production art lives here only after it is reviewed, named, and ready for version control.

Suggested lanes:

- `Furniture`: approved furniture models, materials, and prefabs.
- `Environment`: apartment, balcony, building, city, and prop art.
- `Pets`: pet models, animations, and materials.
- `UI`: source UI art, icons, and exported UI sprites.
- `Materials`: shared production materials and shader-facing assets.
- `Textures`: shared texture libraries.
- `Source`: source-authoring files that are approved for Git LFS.

Keep temporary model drops outside `_Project/Art` until they are normalized. Heavy files require Git LFS or private asset storage.
